PrepPal Test
===================================

**PrepPal** is a web application for people to find and share recipes online.
that creates recipes mixing random ingredients.
It pulls data from the `Open Food Facts database <https://world.openfoodfacts.org/>`_
and offers a *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.

Project Overview
----------------

The Meal Prep App is a mobile application designed to simplify meal planning and preparation. Our application helps users:

* Create and manage their personal recipe collection
* Quickly access their favorite meals
* Streamline their grocery shopping and meal prep process

Scope
-----

The current version of the app focuses on:

* User account creation and management
* Recipe creation and storage
* Basic meal planning functionality

Key Features
------------

* Simple account creation using email and username
* Intuitive recipe creation interface
* Clean, user-friendly design
* Cross-platform mobile compatibility

Target Audience
---------------

Our app is designed for:

* Busy professionals who want to plan meals efficiently
* Health-conscious individuals tracking their nutrition
* Cooking enthusiasts who want to organize their recipes
* Families managing weekly meal plans

Getting Started
---------------

To begin using the Meal Prep App:

1. Download the app from the App Store or Google Play
2. Create an account with your email and username
3. Start adding your favorite recipes

For developers interested in contributing, see our :doc:`development` guide.
