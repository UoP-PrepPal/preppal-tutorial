PrepPal General Information 
===================================

**PrepPal** is an application for people to find and share recipes online.
that creates recipes mixing random ingredients.
It pulls data from the `Open Food Facts database <https://world.openfoodfacts.org/>`_
and offers a *simple* and *intuitive* API.

PrepPal Scope
==================================
The Meal Prep App is a mobile application designed to help users plan, organize, and prepare meals efficiently. It simplifies meal planning by allowing users to:

Create and manage recipes – Store ingredients, cooking steps, and estimated preparation time.

Save personal recipes – Users can add their own recipes for quick access.

User-friendly interface – A clean, intuitive design for easy navigation.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
